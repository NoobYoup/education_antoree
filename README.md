## Công nghệ sử dụng

-   Vite (khởi tạo dự án)
-   React + Bootstrap + SCSS (UI/UX)
-   Dummy JSON (fake api)

---

## Hướng dẫn build và run dự án

```bash
# 1. Clone dự án về
git clone https://github.com/NoobYoup/education_antoree.git
cd education_antoree

# 2. Cài đặt dependencies
npm i

# 3. Chạy dev server
npm run dev

```
